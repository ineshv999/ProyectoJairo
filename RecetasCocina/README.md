Propuesta 1: Página Web de Recetas
Factibilidad técnica:
El equipo cuenta con conocimientos básicos en desarrollo web, bases de datos y programación, lo que permite implementar el sistema sin dificultad. Las tecnologías necesarias son ampliamente conocidas y cuentan con abundante documentación.
Factibilidad operativa:
El sistema es sencillo de usar y no requiere capacitación especializada. Cualquier usuario puede interactuar con la plataforma de manera intuitiva.
Factibilidad económica:
El proyecto no requiere inversión económica, ya que se utilizarán herramientas gratuitas y software de código abierto.

Propuesta 2: Inventario de una Tienda Pequeña
Factibilidad técnica:
El desarrollo es viable con los conocimientos actuales del equipo en programación web y bases de datos. La lógica del inventario es clara y no presenta complejidad elevada.
Factibilidad operativa:
El sistema mejora el control de productos y reduce errores manuales. Es fácil de usar y se adapta a tiendas pequeñas.
Factibilidad económica:
No se requieren costos adicionales, ya que se utilizarán recursos gratuitos y el proyecto es académico.

Propuesta 3: Agenda Personal Web
Factibilidad técnica:
El proyecto es totalmente viable con los conocimientos básicos del equipo. El manejo de fechas y eventos es común en aplicaciones web y cuenta con amplio soporte técnico.
Factibilidad operativa:
La agenda facilita la organización personal del usuario y su uso es intuitivo, sin necesidad de capacitación.
Factibilidad económica:
El desarrollo no implica costos económicos, ya que se utilizarán herramientas gratuitas y entornos locales.
